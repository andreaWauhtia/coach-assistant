# 1vs1 technique / duel

- Plan : 25 minutes (prévu)
- Description : Jeu individuel où le joueur effectue une série de sauts/proprioception entre plots, puis part en dribble pour fixer et décaler le défenseur avant de finir l'action. Déroulé exemple : sautillements entre plots -> saut dans 1er cerceau -> saut dans 2e cerceau -> prise d'information -> percussion et finition.
- Objectifs : travailler la fixation, la percussion, la finition et l'anticipation défensive (défenseur serre le côté intérieur).
- Progressions / variantes : imposer pied faible sur la finition, limiter prises de touches, transformer en 1vs1 avec temps limité.
- Notes séance 2026-01-26 : organisé par Steve en groupes de 3; intensité globalement bonne et participation engagée.
- Source : visuel fourni (1vs1 technique/duel)
