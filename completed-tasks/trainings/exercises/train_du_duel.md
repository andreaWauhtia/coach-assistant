# Train du duel / échauffement

- Plan : 10 minutes (prévu)
- Description : Les joueurs courent à rythme moyen derrière le coach. Au coup de sifflet, le dernier de la file doit remonter le wagon en zigzaguant entre les joueurs. Objectif : travailler le départ fulgurant dans un espace très restreint et la capacité à redémarrer rapidement.
- Variantes : raccourcir les relais, ajouter contrainte (passer l'épaule devant le vis-à-vis), intégrer cerceaux pour proprioception.
- Points d'observation : attention rapidement diminuée si l'exercice trop long ou coach à trop haute intensité.
- Notes séance 2026-01-26 : exercice abrégé à 3'30" (prévu 10') car les joueurs perdaient l'attention; premier relais encourageant; apprentissage et appropriation du geste "passer l'épaule".
- Source : visuel fourni (train du duel / échauffement)
